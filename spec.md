仕事できるようになるアプリ 仕様書
1. アプリ概要
項目
内容
アプリ名
仕事できるようになるアプリ（仮）
対象ユーザー
個人用
目的
毎日の仕事を振り返り、できるようになったことを達成感を味わいながら管理し、仕事ができるようになる
技術
HTML + CSS + JavaScript（1ファイル）/ localStorage でデータ保存


2. 画面一覧
	•	ホーム画面
	•	START画面（チェック画面）
	•	カレンダー画面
	•	リスト画面
	•	通知画面

すべての画面下部に「ボトムナビゲーション」（ホーム / カレンダー / リスト / 通知）を表示する。


3. 各画面の詳細
① ホーム画面
上から順に以下を表示する。

	•	今日の目標
	•	リスト画面で☆マークにチェックした項目を表示
	•	できる・達成
	•	1週間連続で「⭕」にチェックした項目を表示
	•	重点項目
	•	「❌」にチェックした項目を、ワースト順（❌が多い順）に表示
	•	STARTボタン
	•	押すとSTART画面（チェック画面）に移動
	•	ボトムナビゲーション
	•	ホーム / カレンダー / リスト / 通知


② START画面（チェック画面）
	•	リスト画面の項目を1つずつ表示し、「⭕」「❌」でチェックする
	•	各項目にコメントを入力できる
	•	⭕を押したら「勝ち・楽勝」を表示
	•	❌を押したら「勝者はすぐに前を向く」を表示
	•	表示形式：1問ごとに切り替わる、または下に質問が続くスタイル
	•	チェック内容はその日の記録として保存する


③ カレンダー画面
	•	カレンダーを表示する
	•	記録した日には⭕マークがつく
	•	日付をタップすると、その日のチェック内容が見られる


④ リスト画面
	•	START画面で表示される質問項目の一覧
	•	各項目は「大きい太字のタイトル」＋「下に通常フォントの補足」で構成
	•	☆マーク：今日の目標として設定できる
	•	一番下の「＋」ボタンで項目を追加できる

6. 中心機能（最初に作るもの）
START画面でリスト項目を⭕❌でチェックし、コメントを書いて、その日の記録として保存する機能。

この機能がアプリの心臓部。ここが動けば、ホームやカレンダーは記録を「見せる」だけの画面になる。
